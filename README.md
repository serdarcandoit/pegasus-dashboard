# Pegasus (PGSUS) Dashboard

This is a professional-grade, interactive stock market dashboard for Pegasus Airlines (PGSUS.IS). The dashboard is built using Streamlit, yfinance, and Plotly, designed to achieve a precise, terminal-like UI/UX that aligns with TradingView's aesthetic.

## Features

- **Real-Time Data**: Fetches up-to-date market prices and historical data via Yahoo Finance.
- **Interactive Charts**: Supports Candlestick and Line charts with smooth interactions.
- **Technical Indicators**: Accurate moving averages calculated locally.
- **Fundamental Analysis**: Essential financial metrics (P/E Ratio, P/B Ratio, Market Cap) displayed elegantly.

## Requirements

- Python 3.8+
- Streamlit
- yFinance
- Plotly

## Installation

1. Clone this repository (or download the files).
2. Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Running the App

Run the following command to start the Streamlit application:

```bash
streamlit run app.py
```

The application will launch in your default web browser.
